This is the MARS Websuite!

This software lets you take all the necessary steps, to run a simulation inside the MARS LIFE simulation system.

Requirements to use this software:

  • A compatible MARS Model written with the MARS Framework in ".NET Core".
  • Data for the simulation in one of the following formates:
    • Geo-potential-field
    • Grid-potential-field
    • Obstacle-layer
    • Table-based
    • Time-series
    • GIS
